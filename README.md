# 艾瑞旺(ERYONE)3D打印耗材RFID数据协议

本仓库定义了一套ERYONE用于在NFC/RFID标签（NATG213）上存储3D打印机耗材参数的标准数据格式。

---

## 协议规范

### 数据格式
- **编码**：UTF-8 纯文本。
- **结构**：
  - 以 `{` 开头
  - 以 `}` 结尾
  - 字段之间用 `;`（分号）分隔
- **语言**：字段内容无限制，建议使用英文以保证广泛兼容性。

### 字段顺序及说明
| 序号 | 字段名（中文） | 字段名（英文） | 说明 / 示例 |
|------|----------------|----------------|----------|-------------|
| 0    | 头             | Header         | 固定为 `{` |
| 1    | 厂商           | Manufacturer   | 例如 `ERYONE` |
| 2    | 耗材主类型     | Material Name  | 例如 `PLA`、`ABS`、`PETG` |
| 3    | 耗材子类型     | Material Supplement | 若无子类型，填一个空格 ` `（半角空格），不可为空 |
| 4    | 颜色           | Color          | 6位十六进制RGB值，例如 `000000` 表示黑色，`FF00FF` 表示紫色 |
| 5    | 透明度         | Transparency   | 2位十六进制，`00` = 完全透明，`FF` = 完全不透明 |
| 6    | 打印最小温度   | Min Temperature | 单位：°C，例如 `190` |
| 7    | 打印最大温度   | Max Temperature | 单位：°C，例如 `230` |
| 8    | 热床温度       | Bed Temperature | 单位：°C，例如 `60` |
| 9    | 直径           | Filament Diameter | 存储为整数，实际值需 **除以100** 得到毫米，例如 `175` = 1.75mm |
| 10   | 重量           | Filament Weight | 单位：克（g），例如 `1000` |
| 11   | 生产日期       | Production Date | 格式：`YYMM`，例如 `2602` 表示 2026年02月 |
| 12   | 尾             | Tail           | 固定为 `}` |

### 完整示例
```
{ERYONE;PLA; ;000000;FF;190;230;60;175;1000;2602}
```
**解析含义**：
- 厂商：ERYONE
- 主材料：PLA
- 子类型：无
- 颜色：黑色（#000000）
- 透明度：完全不透明
- 打印温度范围：190°C – 230°C
- 热床温度：60°C
- 直径：1.75mm
- 重量：1000g
- 生产日期：2026年2月

## 使用说明
- 耗材子类型字段**必须至少包含一个字符**；若无子类型，请使用一个半角空格 ` `，而不是空字符串。
- 颜色和透明度均以十六进制字符串存储，**不包含** `#` 前缀。
- 直径存储为整数，代表实际值的**百分之一毫米**（即毫米×100），以避免文本中出现小数点。
- 生产日期格式 `YYMM`
